Audio Visualizer

about this project.

In this class project, I used the Web Audio API to transform audio frequency and waveform data into dynamic visual elements using the HTML5 canvas. By setting up an audio graph with different nodes for processing and analyzing sound, I then extracted meaningful audio data and mapped it to visual outputs in real time. I also implemented interactive controls through event listeners, allowing users to tweak and manipulate the audio and see how it affects the visuals.

I was extremely excited to work on this project because I've always loved music, and I used to watch those EDM audio visualizer videos on YouTube all the time. Getting to create something like that myself was super fun and nostalgic. I eventually want to keep building on this project in the future, adding more personalized features and playful elements to give it a little bit of my own personality.
