FEVERISH

about this project.

FEVERISH is a short, choice-based adventure game developed in Twine, based on a short story I originally wrote for a creative writing class. It is set in a world I've been independently building outside of classes. Players must keep an eye on resources like health, fever, and inventory items as they navigate the story. Player choice will determine one of three distinct endings: a good, bad and TRUE ending.

I created this project as a way to expand my skills in narrative design and interactive storytelling. Writing and storytelling are a big part of what I want to do in game development, and this project was a great opportunity to help develop that further. I have also linked the short story set in the same world, which helps provied additional worldbuilding context.
